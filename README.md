# Superstore Sales Analysis (Data Analytics Project)

## Objective
The objective of this project is to analyze Superstore sales data and identify
key factors affecting sales and profitability to support business decisions.

## Dataset
The dataset contains retail order-level data including sales, profit, discount,
product categories, and regional information.

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data cleaning and preprocessing
- Category-wise sales and profit analysis
- Region-wise sales performance analysis
- Discount impact on profit analysis
- Exploratory Data Analysis (EDA)

## Key Business Insights
- Technology category generates the highest profit.
- Furniture category shows lower profitability despite reasonable sales.
- Higher discounts negatively impact profit margins.
- West region performs best in terms of total sales.

## Conclusion
This project demonstrates my ability to clean data, perform exploratory data
analysis, and derive actionable business insights using Python for data analytics.
